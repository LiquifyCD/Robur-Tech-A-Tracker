# Fondkoll

Fondkoll is a responsive fund search, tracking, watchlist, and comparison app. It began as a single-fund Swedbank Robur Technology A intraday estimator and now supports fund discovery by name, provider symbol, and—when the upstream source recognises it—ISIN.

The deployable project is in [`robur-tracker/`](./robur-tracker/README.md).

## Important

Values are informational and may be delayed. The app is not financial advice and does not replace the fund company's official NAV. See the project README for data licensing, accuracy, privacy, and operational limitations.
